[view engine에 대해서]
=> express 는 기본적으로 process.cwd() + /'views' 파일 경로로 view 파일을 찾아 클라이언트에게 던진다.
=> configuration 을 통해서 path 를 원하는 곳으로 변경 가능하다.
=> pug (과거 jade)를 안쓰기 위해서는 ejs view engine 을 사용하면 된다. app.set("view engine", "ejs");
